# Assembly #

1. Install output bearing into front housing with retaining compound
2. Install internal gear in front housing with retaining compound
3. Install outer housing onto front housing
4. Install shaft bearing into planet output with retaining compound
5. Install planet output into output bearing
6. Install sun gear into sun gear holder using heat and cold to make a shrink fit
7. Install sun gear holder onto rotor
8. Press input bearing into planet input
9. Install planet bushings into planets
10. Press needle bearings into planet bushings
11. Drop planet shafts into planet output while resting on back
12. Slide planets onto shafts, with bushing fat end near planet output
13. Install planet input, press onto planet shafts, then use M3 bolts to attach
14. Install stator onto front housing using cold for a shrink fit
15. Press M4x24 dowel into sun gear holder / sun gear until it hits
    back of sun gear
16. Install rotor into gearbox.  Use M3 dowel to press shaft into
    front shaft bearing
17. Align rotor and stator using paper strips
18. Pick @!!@#% shards of metal out of rotor using microscope
